# knapsack_greedy
knapsack_greedy.h — описаны структуры данных и объявления функций
knapsack_greedy.cpp — реализованы чтение входного файла, сортировка предметов по отношению ценности к весу, сам жадный выбор предметов и вывод результата.
main_greedy.cpp — запуск программы через командные аргументы, чтобы можно было передавать имя входного файла и запускать алгоритм на разных тестах без изменения кода.

Жадный алгоритм для задачи о рюкзаке довольно простой. Надо выбирать предметы по максимальной удельной ценности, т.е. по отношению value / weight. Такой подход работает очень быстро, так как требует только сортировки предметов и одного прохода по ним. По итогу:
- жадный алгоритм является самым быстрым и самым простым в реализации;
- локальный поиск обычно даёт более качественное решение, чем простой жадный алгоритм;
- метод ветвей и границ даёт точное оптимальное решение, но может быть существенно тяжелее по времени;
- генетический алгоритм является эвристическим методом, который обычно сложнее жадного, но может находить более хорошие решения за счёт более широкого поиска.
Жадный алгоритм удобно использовать как быстрое базовое приближение, однако по качеству решения он обычно уступает точным и более сильным эвристическим алгоритмам.
